# AIAG & VDA Reference-Aligned FMEA App

This is a static browser app for creating and maintaining a Failure Modes & Effects Analysis worksheet. It is designed to be hosted from GitHub Pages and linked from an existing website.

## Use

Open `index.html` in a browser. The app saves work locally in the browser and can export or import project data as JSON. It can also export the worksheet as CSV.

## GitHub Pages Hosting

1. In GitHub, open **Settings > Pages**.
2. Choose **Deploy from a branch**.
3. Select the `main` branch and `/root`.
4. Use the published Pages URL as the link from your website.

## Reference Notice

This app is a working aid based on publicly described AIAG & VDA FMEA concepts. It does not reproduce the AIAG & VDA FMEA handbook. Confirm scoring tables, Action Priority decisions, and customer-specific requirements against your licensed handbook and applicable OEM requirements.
